CACBBH

Cellular Automata Classifier using Binary Black hole optimization, is a new approach for data classification based on the cellular automata rules.

What is a Cellular Automata

The cellular automata algorithm takes a series of states (binary states in the elementary version) and produces the next generation of this sequence, and by using specific set of rules we can change the behavior of the changing states to suite our aim to find the best solution, to ultimately get the best solution in the best time.!
